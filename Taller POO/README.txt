Instrucciones para ejecutar y probar la Aplicación de Gestión de Empleados
Esta aplicación de gestión de empleados está desarrollada en C# y utiliza conceptos de Programación Orientada a Objetos (POO). Permite gestionar información sobre empleados y gerentes, calculando y mostrando detalles como el salario mensual, salario anual y el departamento (en caso de gerentes).

Requisitos Previos
Asegúrate de tener instalado:

.NET SDK
Pasos para Ejecutar la Aplicación

bash

Compilar la Aplicación:

bash
Copy code
dotnet build
Ejecutar la Aplicación:

bash
Copy code
dotnet run
Cómo Probar la Aplicación
La aplicación solicitará información sobre un empleado y un gerente. Sigue las instrucciones en la consola para ingresar los datos requeridos.

La aplicación mostrará la información ingresada y calculará el salario anual.

Repite el proceso para probar con diferentes empleados y gerentes.
